## Project Overview

Finance Planner - A Python-based personal finance planning application (early stage).

## Environment Setup

This project uses UV for Python package management.

**Requirements:**
- Python >= 3.12
- UV package manager

**Setup:**
```bash
uv venv
source .venv/bin/activate  # On macOS/Linux
uv pip install -e .
```

## Running the Application

```bash
python main.py
```
